<h1> 🎉 경기도 문화행사 탐색부터 동행 모임까지, GyeonggiFesta </h1>

**GyeonggiFesta**는 경기도 전역의 문화행사·축제를 쉽고 빠르게 탐색하고,  
관심사 기반 AI 추천과 **동행찾기 모임팟** 기능을 통해  
함께 즐길 사람과 연결되는 **문화 여가 통합 플랫폼**입니다.

사용자 취향에 맞춘 추천, 실시간 행사 정보 제공, 동행 모집 기능을 통해  
경기도의 문화 경험을 더 풍부하고 쉽게 누릴 수 있습니다.

## 🎯 개발 목적
- 단순 행사 나열을 넘어 **취향 기반 추천 + 동행 참여까지 이어지는 통합 플랫폼 구축**
- **1인 가구·신규 지역 거주자**의 여가 참여도 향상  
- 경기도의 다양한 문화자원을 더 많은 사람들이 함께 즐길 수 있는 생태계 조성  
- 지역 기반 커뮤니티 활성화 및 문화 접근성 향상  

<p align="center">
<b>“경기도 문화행사 탐색부터 동행 참여까지, 하나로 연결되는 올인원 플랫폼”</b>

## ✨ 주요 기능

### 🔹 1. 행사 탐색 (홈 화면)
- 현재 진행 중인 경기도 문화행사 실시간 제공  
- 무한 스크롤 기반 탐색으로 행사 정보 접근성 향상  
- 카테고리·일정·키워드 기반 통합 검색 제공  

### 🔹 2. 행사 상세 페이지
- 행사명, 시간, 장소, 요금, 주최자 정보 제공  
- 공식 링크, 운영 시간, 기본 주차 정보(위치·요금·운영시간) 표시  
- 즐겨찾기(관심 행사) 등록 기능  

### 🔹 3. AI 기반 개인화 추천
- 검색 이력·즐겨찾기 데이터를 조합한 추천 모델 적용  
- 경기 지역 행사 취향 기반 큐레이션 목록 제공  
- 실시간 배치 잡으로 추천 데이터 자동 갱신  

### 🔹 4. 동행찾기 모임팟 모집 기능
- 행사 참여를 위한 ‘동행 팟’ 생성  
- 파티원 모집글 작성 
- 멤버 간 채팅방 자동 생성  
- 일정 캘린더 자동 등록 기능 제공  

### 🔹 5. 접근성 중심 사용자 경험
- 위치 기반 행사 탐색  
- 사용자 맞춤형 UI 흐름(카드형 리스트·탭 UI 등)  
- 모바일 중심 반응형 구성

## 🛠️ 기술 스택

### Backend
- Spring Boot
- JPA / QueryDSL  
- MySQL (RDS)  
- Redis (세션/토큰 캐싱)  
- Spring Batch (행사 데이터, AI 추천 데이터 배치)  
- RestTemplate / WebClient (공공데이터 연동)

### Frontend
- React + TypeScript / SPA, Vercel

### Infra & DevOps
- AWS EC2 (Ubuntu 24.04)  
- AWS RDS(MySQL), ElastiCache(Redis)  
- AWS S3 (이미지 업로드)  

### External APIs
- 경기도 문화행사 공공데이터 API

## 🤖 AI 추천 모델

경기페스타는 사용자 검색 패턴·즐겨찾기 행사 데이터를 기반으로  
사용자와 가장 연관성이 높은 행사를 추천합니다.

- 배치 기반 추천 데이터 생성 (`Spring Batch`)  
- 외부 ML 서버(Koyeb)에 요청하여 추천 목록 생성  
- 추천 결과를 Redis와 DB에 저장해 빠른 응답 제공
